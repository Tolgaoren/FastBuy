# FastBuy
E-commerce app
## Kullanılan bileşenler
- Clean Architecture
- Mvvm
- Hilt
- Jetpack Compose
- Xml
- Fragment Navigation
- Retrofit
- Room
- Encrypted Shared Preferences
- Coroutines
- Livedata
- Push Notification
- Firebase Remote Config
- Glide
- Gson
- Threenabp
